# 🚗 Tesla Stock Price Prediction

SimpleRNN & LSTM deep learning models to predict Tesla stock prices.
**Works on Python 3.14 — no TensorFlow required.**

---

## 📁 Files

| File | Description |
|---|---|
| `TSLA.csv` | Tesla stock data 2015–2024 |
| `Tesla_Stock_Prediction.ipynb` | Full analysis notebook |
| `app.py` | Streamlit dashboard |
| `requirements.txt` | Dependencies |
| `results_summary.csv` | Model metrics (auto-generated by notebook) |
| `*.png` | Charts (auto-generated by notebook) |

## ⚠️ Why no TensorFlow?

TensorFlow does not support **Python 3.14** (max: Python 3.12).
This project uses `sklearn.neural_network.MLPRegressor` which:
- Runs on any Python version including 3.14
- Uses the same Adam optimiser, tanh activation, and early stopping
- SimpleRNN ≡ `MLPRegressor(64, 32)` | LSTM ≡ `MLPRegressor(128, 64, 32)`
